Основным инструментом для стилизации в проекте является **Tailwind CSS**. Использование чистого CSS (в файлах `.css`) допускается только в нескольких исключительных случаях.

### Исключения

- **Медиа-запросы:** Для создания адаптивных стилей, которые зависят от высоты или ширины экрана.
- **Динамические значения через `clamp`:** Когда требуется плавное изменение размера (например, шрифта) в зависимости от ширины экрана.
- **Стилизация дочерних элементов внешних библиотек:** Когда нет прямого доступа к DOM-элементам компонента из внешней библиотеки (например, Highcharts).
