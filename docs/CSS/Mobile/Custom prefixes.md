Для адаптивной вёрстки в проекте используются три кастомных префикса. Они работают в связке с медиа-запросами или CSS-переменными.

| Префикс       | Использование | Назначение                                                    |
| ------------- | ------------- | ------------------------------------------------------------- |
| w (wsm)       | media-query   | Применяет стили на основе **ширины** экрана                   |
| h (hsm)       | media-query   | Применяет стили на основе **высоты** экрана                   |
| vp (viewport) | classes       | Используется для классов с динамическими значениями (`clamp`) |

> 💡 **Лучшая практика**
> Для динамических значений `clamp` следует использовать отдельные переменные, если значение используется в нескольких местах, тогда выносить его в (`:root`)

**Пример:**
```css
@media (min-width: 25.8125rem) /* 413px */ {  
    .wsm\:leading-9 {  
        @apply leading-9;  
    }  
}

@media (min-height: 41.75rem) /* 668px */ {  
    .hsm\:inline-flex {  
        @apply inline-flex;  
    }  
}

.vp\:visual-bottom {  
    --dynamic-bottom: clamp(9.5625rem, calc(19.93rem - 38.57vw), 11.25rem);  
    @apply bottom-[var(--dynamic-bottom)];  
}
```
