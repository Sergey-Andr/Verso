При решении фичи могут возникать неочевидные решения (магические числа внутри JSX, хаки в CSS и т.д.).  
В этом разделе собраны пояснения к таким моментам для облегчения поддержки старых решений.

---

## Как добавить новый файл
1. Создаётся отдельный `.md` под компонент или функцию.  
2. В начале файла указываются:
   - путь к исходному файлу в проекте,
   - краткое описание (зачем нужна фича).  
3. Далее идёт раздел **Магические решения** — список всех нестандартных чисел/решений с пояснениями.
