**Файл:** `app/(pages)/Home/components/mobile/SettingsDesktop/components/CloudsTrackAnimation.tsx`  
**Описание:** Анимация на странице настроек. Используется для визуального заполнения пустоты.

---

## Магические решения

### Отступы

- **`-ml-6`**  
  Используется для компенсации падинга страницы настроек.  
  Начало анимации сдвигается за пределы экрана.  
  (Аналогично работает `calc(100% + 1.5rem)`).

### Длина трека

- **`400%`**  
  Длина дорожки анимации.  
  Выбрано для создания достаточно длинного цикла, чтобы два блока облаков анимировались без шва.

### Позиционирование

- **`top-7/12` и `top-8/12`**  
  Смещение трека на маленьких экранах.  
  Подобрано для центрирования облаков.

### Размеры облаков

- **`w-30dvw`**  
  Оптимальный размер облаков: не слишком большие и не слишком маленькие.
- **`w-27dvw`**  
  Нижний трек уменьшен на ~10% для создания эффекта параллакса.

### Анимация

- **`translateX(-54%)` (scrollClouds)**  
  Ключевой стиль для бесшовной анимации.  
  Меньшее или большее значение приведёт к разрывам между облаками.
