**Цель:** Упростить набор иконок и сделать отображение единообразным: близкие по смыслу WMO-коды показываем одной и той же иконкой.

**Источник:** Open-Meteo `weather_code` (WMO).

**Инварианты маппинга (объединения):**

- `48 → 45` — оба описывают туман (с инеем/без), используем одну иконку «fog».
    
- `51, 53 → 61` — морось и слабый дождь показываем как «лёгкий дождь».
    
- `55 → 65` — плотная морось = «сильный дождь».
    
- `56 → 66`, `57 → 67` — переохлаждённая морось маппится на ледяной дождь (слабый/сильный).
    
- Остальные коды отображаются 1:1.
    

**Как расширять:**

1. Появился новый WMO-код → определить группу (дождь/туман/снег/гроза) и добавить в `WEATHER_ICON_MAP`.
    
2. Если нет подходящей иконки — добавить файл в `public/weather` и сослаться на него в маппинге.
    
3. Обновить доку (этот файл).
    

**Замечания:**

- На уровне `next/image` формат может конвертироваться автоматически (webp/avif), исходники храним в одном формате (`png`) для простоты.
